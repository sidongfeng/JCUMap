JCUMap Subject Mapping Tool 1.0

Developed by Dr David Holmes, Copyright 2014

Software created at James Cook University (JCU), funded in-part by a JCU Teching and Learning Development grant.

This tool provides a guided method by which to map each subject in a degree program against the appropriate Competency Standards or Program Learning Outcomes (PLOs). It produces an xml file per subject that can then be added to a program database for compilation and analysis. This mapping can be used for internal program management, benchmarking, and for accreditation and regulatory purposes. At any point during the subject mapping, work can be saved and returned to at a later time. Progress is saved in a second xml project file that can be reopened later to continue the mapping. When updating a subject, this file can be reopened and updated to generate a new result file as well.

To begin, either open an existing project, or start a new one.
